## 项目使用

安装git，在需要使用的磁盘目录下右键 `Git Bash Here`，粘贴 `git clone git@github.com:bucengfangqi/e-zorro.git`，使用[vsCode](https://code.visualstudio.com/)打开项目，并运行 `npm install`即可开启项目
